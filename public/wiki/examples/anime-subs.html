Styled Subtitles

Advanced ASS/SSA subtitle rendering using the OctopusPlugin WebAssembly engine. Custom fonts are loaded from a remote fonts.json manifest — no local files needed.

Rail Wars!

2014 · Anime · OctopusPlugin Demo
The opening sequence features styled Romaji and English lyrics rendered via ASS subtitles with custom positioning, colors, and the "Negotiate Free" font loaded at runtime.

Playback

Subtitles

How it works: The OctopusPlugin uses libass compiled to WebAssembly to render ASS subtitles pixel-perfectly on a canvas overlay. A fonts.json manifest tells the player which custom fonts to fetch, so styled typographic effects render correctly even without the font installed on the viewer's machine.
