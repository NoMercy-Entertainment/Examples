Custom Storage (IndexedDB)

Wiki ↗

A custom StorageInterface backed by IndexedDB instead of the default localStorage. The player uses this adapter for all internal storage (volume, subtitle preferences, audio language, etc.). Every get, set, and remove call is logged below so you can see exactly what the player stores. Use the buttons to manually exercise each storage operation.
